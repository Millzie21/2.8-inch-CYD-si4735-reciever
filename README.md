
# 2.8-inch-CYD-si4735-reciever
Modification of https://github.com/IU4ALH/IU4ALH/blob/main/SI4735_2.8_TFT_SI5351_V.5.2b.zip to run on 2.8 cheap yellow display, original credit goes to him not me as I did not make the firmware I just make it work on the cheap yellow display. Library lovyanGFX is required, install from arduino library manager and the buttons are red instead of blue to easily tell. I havent recieved the si4732 module yet which is why it shows it not connected on boot screen
